# Mood Analyzer

Esse projeto surgiu da vontade de criar algo diferente dos exercícios tradicionais e que tivesse um pouco mais de personalidade. Em vez de seguir um padrão comum, a ideia foi desenvolver um programa simples, mas que interagisse com o usuário de forma mais humana.

O Mood Analyzer é um sistema em C que analisa um texto digitado pelo usuário e tenta identificar o humor com base em palavras presentes na mensagem. A partir disso, o programa calcula uma porcentagem aproximada entre diferentes estados emocionais, como felicidade, tristeza e estresse.

Além da análise, o sistema também retorna uma mensagem motivacional de acordo com o resultado, buscando criar uma experiência mais acolhedora. Outro ponto importante foi a implementação de um histórico, onde cada entrada é salva em arquivo, permitindo acompanhar interações anteriores.

Durante o desenvolvimento, o foco principal foi praticar manipulação de strings, lógica de decisão e uso de arquivos, mas também pensar em como tornar a interação menos mecânica e mais próxima de algo real.

Este não é um sistema complexo do ponto de vista técnico, mas representa um avanço em relação a projetos básicos, principalmente por trazer uma proposta diferente e mais criativa. A intenção foi sair do padrão e explorar uma ideia que, mesmo simples, demonstra iniciativa e capacidade de pensar além do convencional.

Esse projeto faz parte do meu processo de aprendizado em desenvolvimento de sistemas, onde busco evoluir não apenas na parte técnica, mas também na forma de construir soluções que tenham algum significado.
